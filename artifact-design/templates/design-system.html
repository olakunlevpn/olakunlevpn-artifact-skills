06

Voice & density rules

  • One loud color only — clay is the accent. Olive / rust / sky are reserved for true semantic signals.
  • Serif headings, sans body, mono for small bits — never invert these.
  • 1.5px borders — not 1px. The slightly heavier stroke is the visual signature.
  • Generous whitespace — page padding 56px 24px 120px is intentional. Don't compress.
  • No emojis as icons — use SVG, a .dot, or a .badge instead.
  • No drop shadows by default — reserved for hover and floating elements.
