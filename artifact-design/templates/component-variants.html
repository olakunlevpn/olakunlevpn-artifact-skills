When to use

Reach for this when

  • You have the single most important action on a page (use .button-primary).
  • You have a routine action — save, submit, continue (use .button).
  • You have a tertiary action that should not pull focus (use .button-ghost).

Don't reach for this when

  • You need a navigation link that's part of running prose — use a regular <a>.
  • You need a destructive action — pair the button with a confirmation dialog or use .button-ghost with explicit copy.
  • You have more than two CTAs in a single section — back off one to ghost, otherwise the page has no focus.
